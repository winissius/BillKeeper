# BillKeeper

This project is a simple bill management system built with FastAPI, MongoDB and Docker.

## Features

* **Create, read, update, and delete bills:** Manage your bills efficiently with a user-friendly API.
* **MongoDB database:** Store bill data in a scalable and flexible database.
* **Dockerized environment:** Easily deploy and run the application in a containerized environment.
* **FastAPI framework:**  Build a fast, efficient, and type-safe API with Python.
* **User authentication:** Secure access to the API using JWT authentication.
